# 📁 Умный Архиватор и Сортировщик (Smart Organizer & Archiver)

Скрипт на Python для автоматического наведения порядка в файловой системе. Он сканирует указанную директорию, группирует все файлы по папкам в зависимости от их расширения, создает резервную копию в виде ZIP-архива с отметкой текущей даты и (по желанию) удаляет исходные данные.

## ✨ Основные возможности

* **Генерация хаоса (Тестовый режим):** Возможность автоматически создать запутанную структуру из случайных папок и пустых файлов разных форматов для безопасного тестирования работы скрипта.
* **Умная сортировка:** Извлечение файлов из любой глубины вложенности и их распределение по папкам вида `txt_files`, `jpg_files` и т.д. Файлы без расширения попадают в `other_files`.
* **Защита от перезаписи:** Если в разных исходных папках находились файлы с одинаковым именем (например, два разных `index.html`), скрипт не перезапишет их, а аккуратно добавит числовой индекс к имени копии.
* **Игнорирование исключений:** Скрипт обходит стороной папки с названием `ignore_me`, оставляя их содержимое нетронутым.
* **Автоматическое резервное копирование:** Упаковка всех отсортированных данных в единый ZIP-архив, в название которого вшита текущая дата (например, `Sorted_Data_data06-03-2026.zip`).
* **Тотальная зачистка:** Опциональное безвозвратное удаление исходной "грязной" папки после успешного создания архива.

## ⚙️ Требования

* **Python 3.12 или выше.** *В скрипте используется современный и элегантный метод `Path.walk()` из библиотеки `pathlib`, который был добавлен в язык начиная с версии 3.12.*
* Никаких сторонних библиотек устанавливать не нужно, используются только встроенные модули Python (`pathlib`, `shutil`, `datetime`, `random`, `sys`).

## 🚀 Как использовать

1. Скачайте скрипт на свой компьютер.
2. Запустите скрипт через терминал или командную строку:
```bash
python main.py

```


3. Скрипт задаст три вопроса в интерактивном режиме:
* **`Введите путь где нужно навести порядок ---->`** Укажите абсолютный или относительный путь к папке (например, `C:\Users\Admin\Downloads\test_folder`).
* **`Удалить ли выбранную папку? Будет создан zip архив в обоих случаях! y/n --->`**
Введите `y` (да), если хотите удалить исходную папку после архивации, или `n` (нет), чтобы оставить её.
* **`Создавать Хаос? y/n ---->`**
Введите `y` (да), если указанная папка пуста и вы хотите наполнить её тестовыми файлами и подпапками перед сортировкой.



## 🛠 Как это работает под капотом

1. Скрипт считывает введенный путь и проверяет его существование с помощью `pathlib`.
2. Если выбран режим хаоса, функция `generate_chaos` создает от 10 до 100 случайных подпапок и файлов.
3. Функция `sort_files` обходит дерево директорий, исключает папку `ignore_me` и перемещает файлы на уровень выше исходной директории в промежуточную папку `Sorted_Data`.
4. С помощью `shutil.make_archive()` промежуточная папка `Sorted_Data` сжимается в ZIP-архив.
5. Происходит удаление временной папки `Sorted_Data` и (если выбрано) исходной целевой директории с помощью `shutil.rmtree()`.

## ⚠️ Важное предупреждение

Функция удаления (при выборе ответа `y` на вопрос об удалении) использует метод `shutil.rmtree()`. Эта команда удаляет файлы и папки **НАВСЕГДА**, минуя системную "Корзину".
**Будьте предельно внимательны** при вводе пути к папке, чтобы случайно не удалить важные системные или личные данные! Рекомендуется сначала протестировать скрипт на безопасной пустой папке, используя функцию "Создания Хаоса".
